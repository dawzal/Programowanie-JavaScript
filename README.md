# ToDo App
Aplikacja na zaliczenie przedmiotu Programowanie JavaScript
Dawid Zalewski 

Prosta aplikacja ToDo stworzona w React + Vite.

## Jak uruchomić

1. Sklonuj repozytorium:
https://github.com/dawzal/Programowanie-JavaScript.git
2. Przejdź do folderu projektu:
cd todo-app
3. Zainstaluj zależności:
npm install
4. Uruchom aplikację:
npm run dev

## Funkcje

- Dodawanie zadań
- Edytowanie zadań
- Usuwanie zadań
- Lista dynamicznie się aktualizuje
